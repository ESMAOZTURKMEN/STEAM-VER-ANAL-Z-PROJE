## 📋 Ödev Özeti ve Bulgular

Bu çalışmada, Steam veri seti üzerinde Pandas, Matplotlib, Seaborn ve Plotly kütüphaneleri kullanılarak tanımsal istatistikler ve görselleştirmeler üretilmiştir. 

* **Veri Seti Yapısı:** Rapor, Steam platformundaki oyunlara ait fiyat, oyuncu sayıları ve kategorik değişkenleri incelemektedir.
* **Yönetici Özeti:** GitHub Pages üzerindeki canlı raporda, bonus puan şartlarına uygun olarak Python kod hücreleri gizlenmiş (`--no-input`), sadece grafikler ve analiz yorumları bırakılmıştır.
